## 1. 앱 내에서 계정 삭제하는 방법

Avarun 앱은 앱 내부에서 즉시 계정을 삭제할 수 있는 메뉴를 제공합니다.

1. Avarun 앱 실행 후 로그인
2. 하단 탭에서 **"프로필"** 이동
3. 화면 우측 상단 **"설정"** 아이콘 탭
4. **"회원 탈퇴"** 선택
5. 확인 다이얼로그에서 **"탈퇴하기"** 탭

별도의 이메일 문의나 승인 과정 없이 **즉시** 계정이 삭제됩니다.

## 3. 삭제되는 데이터
