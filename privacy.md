## 5. 보유 및 이용 기간

- 이용자가 계정을 유지하는 동안 보관합니다.
- 이용자가 **계정 삭제(회원 탈퇴)** 를 요청하면, 아래 데이터가 즉시 삭제됩니다.

  - 달리기 기록 전체
  - 받은 이모지 리액션
  - 참여 중인 팀에서 자동 탈퇴 (본인이 생성자인 팀은 다음 가입자에게 자동 승계되거나, 남은 멤버가 없으면 팀 자체가 삭제)

- 단, 법령에 따라 보존이 필요한 경우 해당 기간 동안 보관할 수 있습니다.

**삭제 경로**: 앱 내 "설정 → 회원 탈퇴" 메뉴. 별도 문의 없이 즉시 처리됩니다.

## 7. 개인정보의 안전성 확보 조치

- 본 앱은 자체 비밀번호를 보관하지 않으며, 인증은 Apple / Google OAuth 제공자에 위임합니다.
- 서버와 앱 간 통신은 HTTPS (TLS) 로 암호화됩니다.
- Supabase Row Level Security (RLS) 를 적용하여, 각 사용자는 본인이 접근 권한을 가진 데이터만 조회·수정할 수 있습니다.
- 서버 측 관리자 권한(서비스 키)은 Edge Function 내부에서만 사용되며, 클라이언트 앱에는 배포되지 않습니다.

## 8. 쿠키 및 분석 / 오류 추적 도구

- 본 앱은 웹이 아닌 모바일 앱이므로 쿠키를 사용하지 않습니다.
- 현재 별도 분석 SDK (Google Analytics, Firebase Analytics 등) 를 연동하고 있지 않습니다.
- **오류 추적**: 앱의 안정성 향상을 위해 Sentry (Functional Software, Inc.) 를 통해 크래시 / 처리되지 않은 예외를 수집합니다.
  - 전송 항목: 사용자 식별자(UUID), 발생 시점, 앱 버전, 익명화된 스택 트레이스
  - 전송하지 않는 항목: 이메일, 닉네임, 러닝 메모, 인증 토큰, 요청 본문 (자동 마스킹 처리)
  - 보관 기간: Sentry 기본 정책 (이벤트당 최대 90일)에 따릅니다.
  - 비활성화 옵션: 현재 인앱 토글은 제공하지 않으나, 계정 삭제 시 사용자 식별자 매핑이 즉시 해제됩니다.
